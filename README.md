## Introduction

一个 wallhaven 网站的看图客户端软件，滑动直接浏览大图

![](./doc/gallary.png)

## 运行

### 安装依赖

```
npm install
or
yarn install
```

### 启动服务

```
npm run dev
```

### 打包客户端

```
npm run electron:build
```
